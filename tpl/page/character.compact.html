Helden aus der Helden Software importieren

Sie können einen Helden aus der Helden Software importieren. Exportieren sie ihn dazu in der Helden Software über den Menü-Eintrag Datei/Exportieren/Held Exportieren. Die Helden Software wird den Helden nun im XML Format an einen von ihnen ausgewählten lokalen Speicherort ablegen.
Danach klicken sie hier auf dieser Seite den Knopf "Datei Auswählen" und wählen sie die gespeicherte XML Datei aus.
Drücken sie nun noch auf "Importieren" und ihr Held wird für das Solo-Abenteuer hochgeladen.
