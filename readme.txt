This plugin allows you to display a character manager for rpgs in your posts using the shortcode [rp-hero][/rp-hero].

== Description ==

You can use this plugin to show an character manager.

For example:

`
[rp-hero name='MyHero'][/rp-hero]
`

== Installation ==

1. Upload <strong>sonnenstrasse-character</strong> folder to the <strong>/wp-content/plugins/</strong> directory.
2. Activate the plugin through the <strong>Plugins</strong> menu in WordPress.
3. That's all.
